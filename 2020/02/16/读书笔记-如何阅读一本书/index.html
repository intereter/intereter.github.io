读书笔记:如何阅读一本书

发表于 2020-02-16 分类于 读书笔记

##最重要的是，你通过阅读得到的知识，可以转化成怎样的措施和生活建议，可以用来做什么和怎么做。关键时阅读完这本书后，以后你会怎么做

  1. 第一章阅读的活力与艺术：
    内容： 阅读分为为了获得咨询阅读或消遣而阅读，还有为了增进理解而阅读。为了增进理解阅读需要我们去主动阅读，主动阅读需要观察力，记忆，分析，想象。主动阅读时一个主动学习的过程。

  2. 阅读的层次

    1. 基础阅读：基本的阅读，明白文中的句子在说什么，要求是能认字就行。
    2. 检视阅读：在有限时间内，快速了解一本书表象上给你的知识。
    3. 分析阅读：深度分析一本书，将一本书变成你自己的。然而要注意的是，有些书值得这么做，有些仅仅值得浅尝辄止。
    4. 主题阅读：在一个主题上阅读多本书

  3. 检视阅读：

    1. 两种检视阅读：有系统的略读或粗读 与 粗浅的阅读
    2. 检视阅读的帮助：告诉你一本书值不值得分析阅读。告诉你一本书的大概。
    3. 有系统的阅读：第一步，先读书名和序，给这本书分类。第二步，读书的目录，在心里留下一个大概框架。第三步，读一读索引和出版者介绍。完成以上三步，可大概知道是否需要继续往下读。第四步，四处翻一翻，留意书的脉络是怎样递进的，绝对不要错过作者最后的总结
    4. 粗浅的阅读：暂时放下不理解的地方。
    5. 什么时候使用检视阅读： 想要大概了解一本书，知道其是否值得读时。阅读一本难懂的书时。

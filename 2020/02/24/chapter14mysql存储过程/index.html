chapter14mysql存储过程

发表于 2020-02-24 分类于 数据库

  1. 什么是存储过程：
    一组复杂的mysql语句的封装，可以被外界调用

  2. 存储过程的创建:

    delimiter 结束符号(为了与;区别开)
    create procedure name(参数列表)
    begin
    sql语句
    end结束符号
    //参数列表：
    //参数模式 参数名 参数类型
    in name type
    //参数模式:
    in: 参数可以作为输入
    out:参数可以作为返回值
    inout:上述结合

  3. 存储过程的调用:

    call(参数列表)

  4. 删除： drop procedure name;

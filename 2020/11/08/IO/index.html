IO

发表于 2020-11-08 更新于 2020-11-10 分类于 java

  1. 什么是IO：
    一种流式的数据输入输出模型。在java中可以分为字节流与字符流，或者同步IO，异步IO

  2. File:
    在java中，File表示的是一个文件或目录。构造器需要传入一个相对路径或绝对路径。
    (路径的表示：.表示当前目录，..表示上级目录;在windows中，用\作为路径分隔符，而在java中是\ ,在linux是/)
    常用方法有createNewFile,delete，mkdir,mkdirs

  3. InputStream: java中的同步字节流，是一个抽象类而不是接口。

    1.申请使用：
    IO是系统中的资源，使用需要申请。在文件不存在时或其他原因下，申请IO可能会有异常。因此要在try…catch…finaly语句下使用IO,保证正确的关闭IO，当然也可以使用java 7的新特性 try(source)

2.常用API： read,返回一个字节表示的int值(0 ~ 255)，当为-1表示读完了
3.实现类：FileInputStream,ByteArrayInputStream

    什么是classpath: jvm读取.class文件的环境变量，是一个路径的集合，决定了在哪寻找.class文件以及寻找的顺序。ide上的classpath是当前工程的bin目录和jar包

  1. 序列化： 让对象变成byte数组，从而可保存到文件中或者传输
    要能够序列化的类必须实现java.io.Serializable接口，其是一个空接口，即标记接口，只需implement即可
    传输对象的IO实现类为ObjectInputStream和ObjectOutputStream

  2. Reader
    以字符为单位的读取流。注意的是，在构造时尽量传入编码格式，防止乱码出现
    read()返回的是字符表示的int,为-1时读完
    read(char[] c)返回读了几个字符的int

StringBuilder和StringBuffer

发表于 2019-08-28 分类于 java

1.StringBuilder速度更快，但不支持线程安全
2.两者都支持原地修改字符串，而不是生成新的对象
public static void main(String args[]){
StringBuilder test = new StringBuilder(“123”);
test.append(“a”);
test.append(4);
System.out.println(test);
test.delete(1,2);//不包括end
System.out.println(test);
System.out.println(test.subSequence(0,1));//不包括end
test.setLength(2);//从前到后截断
System.out.println(test);
System.out.println(test.lastIndexOf(“1234”));
}

Calendar

发表于 2019-08-26 分类于 java 
public static void main(String args[]){
    //Calendar是抽象类，需要实例化
    Calendar calendar = Calendar.getInstance();
    System.out.println(calendar.getTime());
    //get()方法，返回当前calendar实例的类变量
    //calendar的月份从0开始
    System.out.println(calendar.get(Calendar.MONTH));
    //set()方法，修改当前calendar实例的类变量
    calendar.set(Calendar.DATE,27);
    System.out.println(calendar.getTime());
    //getActuaMaximum得到实例当前字段的可能最大值
    System.out.println(calendar.getActualMaximum(Calendar.MINUTE));
}

泛型

发表于 2019-08-25

泛型，即宽广的数据类型，用于使某个类，方法，接口，可以使用不同的数据类型，例：
1.泛型类
修饰符 class name<?>{}
2.泛型方法
修饰符 <?> 数据类型(){
}
3.类型擦除：在使用泛型时，若没有指定类型，则默认为object
4.类型限定： <? extend type>
5.通配符： <? , ?>

面向对象的三个特点

多态：要求：继承，重写，父类引用指向子类对象

继承：构造方法不能被继承，在java中要么使用默认构造方法，要么重载构造方法

重写和重载

发表于 2019-08-25

1.重写(overriding)：

针对于有继承关系的类方法，要求输入参数列表，返回参数类型，访问修饰符不小于父类，不能抛出新的异常（父类引用指向子类对象，则调用子类重写过的方法）

2.重载(overload)：

要求参数列表不同，参数类型可以相同，可以抛出不同异常，修饰符无要求

java语言概述

发表于 2019-08-25

1.特点：面向对象，一次编译，到处运行

2.应用：web, android,嵌入式，软件开发（不常用）

3.java虚拟机： 即JVM， java程序先编译成为字节码，再通过对应不同平台的jvm解释成机器码，实现了一次编译，到处运行

4.java的不同版本：j2se(标准版)，j2ee(企业版)，j2me(微型版，主要用于嵌入式)

5.什么是api: api是一种通用概念，java的api即java本身提供的许多强大的类，也叫做java类库

6.数据类型：3整型(short,int,long),字节型(byte),2浮点型(float,double),布尔型

7.string:字符串，长度不可变，使用”+”是生成新的字符串,常用方法length(),charAt(),是否包含方法：contains()，替换方法:replace(),分割方法:split()

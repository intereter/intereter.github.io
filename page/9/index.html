chapter3拷贝

发表于 2020-01-26 更新于 2020-01-27 分类于 java

拷贝有深拷贝与浅拷贝，浅拷贝仅仅拷贝值（对于值类型）和内存地址（对于引用型），深拷贝拷贝所有的成员变量
浅拷贝： 使用=复制，arraycopy
深拷贝：改写后的clone

class Dog implements  Cloneable{
    String name;
    protected Object clone(){
        Dog dog =null;
        try {
            dog = (Dog)super.clone();
        }catch (Exception e){
        }
        return  dog;
    }
}
class Test2 implements Cloneable{
    int x;
    private Dog dog = new Dog();
    void setDog(String name){
        this.dog.name = name;
    }
    String getname(){
        return dog.name;
    }
    Test2(int x){
        this.x = x;
    }
    protected Object clone() {
        Test2 t = null;
        try {
            t = (Test2) super.clone();
            t.dog = (Dog) dog.clone();
        } catch (Exception e) {
        }
        return t;
    }
}public class TestCopy {

    public  static  void  main(String args[]){
        TestCopy testCopy = new TestCopy();
        //使用=是对于对象的浅拷贝，仅仅复制了对象的内存地址
        Test2 t1 =  new Test2(1);
        Test2 t2 ;
        t2 = t1;
        t2.x = 2;
        //System.out.printf("%d %d",t1.x,t2.x);
        //修改clone方法后实现深拷贝（直接使用未修改的clone,即没有 t.dog = (Dog) dog.clone();）
        //（也是浅拷贝，因为直接使用clone只是复制了对象的成员变量的内存地址）
        Test2 t3 = new Test2(3);
        t3.setDog("3");
        Test2 t4 = (Test2) t3.clone();
        t4.x = 4;
        t4.setDog("4");
        System.out.printf("t3 x: %d dogname %s,t4 x: %d dogname %s",t3.x,t3.getname(),t4.x,t4.getname());
    }
}

add : 添加表中的字段 insert: 添加表中内容
drop: 删除表或删除数据库 delete: 删除某一行
change: 修改字段形式 update: 更新某一行内容

  1. 登录数据库服务器：
    mysql -uroot-p

  2. 显示所有数据库中的库：
    show databases;

  3. 选择数据库：
    use databasename;

  4. 查询当前所在表中的数据：
    select * from excelname;
    select name , id from tablename;
    select distinct fieldname from tablename;//查询表中所有不重复的某字段
    select * from score where degree between 60 and 80;//60到80的闭区间
    select * from score where degree in (85,86,87);//degree为85或86或87的记录
    select * from score order by value1 asc;,value2 desc//升序查询，降序是desc,查询按照value1升序，value2降序（优先：value1）

  5. 显示当前库中所有表：
    show tables;

  6. 创建表：
    create table tablename(各字段
    check(条件)//创建表的约束
    );
    //记录是表中的一行，字段是组成记录的元素

  7. 删除表：

    drop table tablename;

  8. 查看某个表：

    describe tablename;
    select function();//mysql中可以使用select查询字段，函数，表达式（计算表达式），常量

  9. 向表中插入字段：

    //第一种，支持插入多个字段,支持子查询
    insert into tablename values(value1),
    (value2);
    //第二种，不支持同时插入多个字段，不支持子查询
    insert into tablename set 字段1=…,字段2=..

  10. 删除字段：
    delete from tablename where …;//可以roll back,假如删除了自增长的字段，字段添加时从断电开始
    truncate table tablename;//清空表，不能带where,且不能roll back，假如删除了自增长的字段，字段从头开始

  11. 更新字段:
    update tablename set field1=,filed2= … where ….;//修改单表中符合条件的某一行中的某个字段值

  12. 向表结构中添加多个字段：
    alter table tablename add fieldname fieldtype…

  13. or and not

    select count(*) from tablename where…..;//统计

    where用于聚合前查询的条件，having 用于聚合后，having后的条件可以使用聚合函数

  20. 模糊查询：like(); not like();

    like ‘k%’;//查询以k开头的字符串
    like ‘%oo%’;//查询包含oo的字符串
